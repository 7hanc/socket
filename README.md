# Client/Server 建立Socket 連線

# Goal
1. Client 發送 UI 輸入的字串給 Server 。
2. Server 回應 Client 收到的字數及Server端的時間。

# 簡易操作(適用於python3)
執行server.py與client.py    
當client端輸入"bye"時，結束
